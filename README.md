# Dashboard de Performance de Vendas 📊

Este projeto consiste em um ecossistema de Business Intelligence para monitoramento de performance comercial, transformando dados brutos em insights estratégicos para a alta gestão.

## 🚀 Principais Indicadores (KPIs)
- **Faturamento Total:** R$ 7,73 Milhões
- **Meta Global:** R$ 7,49 Milhões (Atingimento de 103%)
- **Ticket Médio:** R$ 1,73 Mil
- **Volume de Vendas:** 4.455 itens comercializados

## 🔍 Análises Desenvolvidas
- **Visão Executiva:** Acompanhamento macro de metas e faturamento mensal.
- **Visão Regional:** Liderança de vendas nos estados de Goiás (GO) e Distrito Federal (DF).
- **Mix de Produtos:** Destaque para a categoria de Eletrônicos (32,63% do faturamento).

## 🛠 Tecnologias Utilizadas
- **Power BI:** Construção dos dashboards e visões interativas.
- **DAX:** Criação de medidas complexas para KPIs e metas.
- **SQL / Excel:** Estruturação e modelagem da base de dados.
